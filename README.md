# Yuri Rewrite

Yuri Rewrite 是一个 Windows 优先的本地桌面工具，用于导入 TXT 小说、调用用户自定义 AI 模型分析文本，并将小说改写成双女主 / 百合版本。

应用采用本地优先设计：小说原文、章节拆分、分析结果、改写稿、一致性资产、模型配置、调用日志、导出设置和输出文件都保存在用户自己的电脑上。

## 功能特性

- TXT 小说导入与章节拆分，支持常见中文网文章节格式，并按批次处理长篇文本。
- 批次级 AI 分析与一致性资产维护，提取人物、关系、地点、伏笔、术语和姓名信息。
- 批次级 AI 改写，将原男女性别叙事改写为双女主 / 百合文本，并重点处理主角姓名、标题、正文、代词、称谓和男性化描写残留。
- 小说级设定管理，支持主角姓名、额外女性化人名、身材、体型、严谨/创意模式和高级设定。
- 分析/改写并发处理，可在设置中选择 10、6、3 或不并发，以平衡处理速度、失败概率和 token 消耗。
- 可选 AI 复检修正，改写完成后可让 AI 再检查姓名、代词、称谓、标题、逻辑和设定一致性。
- 多模型配置，支持 OpenAI 兼容接口和 Gemini，并支持思考模式控制、模型测试和 API Key 本地保存。
- 原文 / 改写稿对比、AI 调用日志、耗时统计和 TXT 导出。
- GitHub 地址、检查更新和最新版 portable zip 下载。

## 技术栈

- 前端：React + TypeScript + Vite
- 桌面端：Tauri v2
- 后端：Rust
- 本地数据库：SQLite / rusqlite
- 图标：lucide-react

## 开发环境

需要先安装：

- Node.js
- Rust
- Windows WebView2 Runtime

安装依赖：

```powershell
npm install
```

启动开发版：

```powershell
npm run tauri:dev
```

仅构建前端：

```powershell
npm run build
```

构建 Windows Release：

```powershell
npm run tauri:build
```

生成便携版 zip：

```powershell
npm run package:portable
```

便携版默认输出到：

```text
portable/YuriRewrite-v0.1.6-windows-x64.zip
```

## 使用说明

1. 打开 Yuri Rewrite。
2. 点击“导入 TXT”选择小说文件。
3. 在“模型配置”中填写 Base URL、模型名和 API Key。
4. 点击“测试模型”确认配置可用。
5. 点击“设定”填写当前小说的基本设定和改写模式。
6. 选择批次后点击“分析”，或使用“一键分析改写”自动处理全文。
7. 分析完成后可查看或编辑一致性资产。
8. 点击“改写”处理当前批次，完成后在“对比”页面检查原文和改写稿。
9. 根据需要导出已完成改写的 TXT。

如果用户在未填写必要设定时点击“改写”或“一键分析改写”，应用会提示先完成设定。没有导入小说时，小说设定不可用。

## 模型配置

Yuri Rewrite 支持 OpenAI 兼容接口和 Gemini。常见 Base URL 示例：

- OpenAI: `https://api.openai.com/v1`
- DeepSeek: `https://api.deepseek.com/v1`
- Moonshot: `https://api.moonshot.cn/v1`
- 其他 OpenAI-compatible 服务：填写供应商提供的 Base URL 和模型名。

模型名需要填写供应商当前提供的具体模型 ID。不同供应商对思考模式、JSON 输出和并发请求的支持不同；如果分析或改写频繁失败，可以降低并发数、关闭思考模式或更换模型。

## 本地数据与隐私

- 应用数据存放在当前 Windows 用户的应用数据目录中。
- 替换程序目录不会删除已导入小说、配置和分析数据。
- API Key 会优先存储到 Windows 安全存储；不可用时回退到本地数据库。
- AI 日志会保存模型输出、思考内容、原始响应和耗时统计，但不应包含 API Key 或 Authorization 信息。
- 删除小说时，会同步删除该小说相关的章节、批次、设定、一致性资产、任务、日志和改写数据。

## 分发说明

给其他 Windows 用户使用时，发送便携版 zip 即可：

1. 用户解压 zip 到任意目录。
2. 双击 `yuri-rewrite.exe` 启动。
3. 首次使用时在软件内配置模型和 API Key。
